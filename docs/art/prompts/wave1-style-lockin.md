before committing to full batches. These become the golden reference set that all

## Evaluation Criteria

After generating all 5 pieces, evaluate each against the style guide quality checklist:

- [ ] Linework quality — clear pen strokes, not smooth digital rendering
- [ ] Crosshatching present for shadow/volume — not grey gradients
- [ ] Pure black and white — no accidental grey tones or color
- [ ] Art Nouveau/Deco decorative language consistent across all 5
- [ ] 1920s period accuracy where relevant
- [ ] Silhouette clear at reproduction size
- [ ] Tone matches — heroic wonder/dread, not cute or grimdark
- [ ] Hands, faces, weapons anatomically acceptable
- [ ] No text, watermarks, or artifacts

**If 3+ pieces pass:** These become the golden reference set. Proceed to Wave 2.
**If <3 pass:** Adjust LoRA stack / prompt language and regenerate. Log what changed.

## LoRA Testing Plan

Generate each piece in 3 variants:
1. **No LoRA** — baseline flux-dev output
2. **manga_sketch at 0.35** — test if it adds useful texture
3. **manga_sketch at 0.35 + chiaroscuro at 0.25** — test contrast boost

Log which variant produces the best pen & ink quality per piece.
